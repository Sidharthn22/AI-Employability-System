# 🚀 AI Employability System

An AI-powered web application that analyzes resumes using ATS (Applicant Tracking System) principles and matches resumes against different job roles to identify missing skills and provide recommendations.

---

##  Features

-  Upload PDF Resume
-  ATS Resume Score Analysis
-  Job Role Skill Matching
-  Match Percentage Calculation
-  Missing Skills Detection
-  Resume Improvement Suggestions
-  Resume Preview
-  Responsive UI
-  Dockerized Application

##  Installation

### Clone Repository

```bash
git clone https://github.com/Sidharthn22/AI-Employability-System.git
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

##  Docker

Run the complete project using Docker:

```bash
docker compose up --build
```
